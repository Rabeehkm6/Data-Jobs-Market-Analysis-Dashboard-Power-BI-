 # Data Jobs Market Analysis Dashboard | Power BI 📊
## Overview 🔍

The Data Jobs Market Analysis Dashboard is an interactive Power BI report that explores the global data job market in 2024.
It provides insights into job demand, salary trends, job types, platforms, and employment benefits across key data roles such as Data Scientist, Data Engineer, Data Analyst, Machine Learning Engineer, and more.

This project demonstrates strong skills in data visualization, business intelligence, and analytical storytelling using Power BI.

![Overview Dashboard](/Images/Dashboard.png)

 ## Project Objectives 🎯

- Analyze job demand trends across data roles
- Compare median yearly and hourly salaries

- Identify high-paying and high-demand job titles

- Visualize global job distribution

- Examine job conditions (WFH, degree requirement, benefits)

## Tools & Technologies 🛠️

- Power BI Desktop

- DAX (Data Analysis Expressions)

- Power Query (ETL)

- Data Modeling

- Microsoft Bing Maps

## Dashboard Structure 📁
### 1. Overview – Data Jobs Dashboard

A high-level summary of the entire data job market.

#### Key KPIs

- 📌 Total Job Count: 479K

- 💰 Median Yearly Salary: $113K

- ⏱ Median Hourly Salary: $47.62

- ⭐ Salary Star Rating: Visual salary competitiveness indicator

### Visuals

- 📈 Job count trend by month (2024)

- 📊 Job count by job title

- 💹 Median yearly vs hourly salary (scatter plot)

- 📋 Job summary table with trends

- 🎛 Job Title slicer with drill-through

### 2. Job Title Filtered View

Dynamic view when a specific job role is selected.

Example – Data Scientist

- 👥 Job Count: 98K

- 💰 Median Yearly Salary: $125K

- ⏱ Median Hourly Salary: $43.03

#### Insights

- Monthly hiring trend for the selected role

- Salary positioning among other roles

![Overview Dashboard](/Images/Filtered.png)

### 3. Drill-Through Page – Job Title Details

Accessible using Ctrl + Click on any job title.

#### Detailed Analysis Includes

- 💰 Salary gauges (yearly & hourly)

- 🏠 Work From Home (WFH): 14%

- 🎓 No Degree Mentioned: 7%

- 🏥 Health Insurance Provided: 16%

- 🌍 Global job distribution (map)

- 🌐 Job platforms (LinkedIn, Indeed, BeBee, etc.)

- 🧱 Job types (Full-time, Contract, Part-time)

- ✅ Full-time roles dominate the market (91%)

![Overview Dashboard](/Images/Drill%20through.png)
 ## Key Insights 📈

Senior and specialized roles command higher salaries

Hiring trends decline toward the end of 2024

- Most jobs:

    - Require a degree

    - Are full-time

    - Are on-site or hybrid

🚀 How to Use

Open the .pbix file in Power BI Desktop

Use the Job Title slicer to filter roles

Ctrl + Click on a job title to drill through

Hover over visuals for detailed tooltips
